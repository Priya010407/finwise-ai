# 💰 FinWise AI — Financial Advisor & Expense Manager

AI-powered personal finance assistant for India.

## 🚀 Features
- 📸 Upload UPI/PhonePe/GPay screenshots to auto-track expenses
- 🤖 AI advisor powered by Groq LLaMA
- 📊 Interactive spending dashboard with charts
- 🇮🇳 Indian-specific: SIP, ELSS, PPF, Section 80C advice
- 💬 Chat-based financial guidance

## 🛠️ Tech Stack
- **Frontend:** React + Tailwind CSS + Recharts
- **Backend:** FastAPI + Python
- **AI:** Groq LLaMA 3.3 70B
- **Deployment:** Vercel + Railway

## ⚡ Quick Start

### Backend
```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```
